Hello!

This is a homemade version of Bop It along with its PCB design and gbr files. 
gbr files can be opened using gerbv software. 
To manufacture, site that was referenced was macrofab. 
Software used to develop PCB Design was EAGLE.
Most components were provided by Erik Welsh.
LCD Display was downloaded from this Github Repository.
 - https://github.com/eez-open/eagle-libraries
Joystick and PocketBeagle symbol components was created by me.  
Folder includes Library used, Schematics, and Layout.
Manufacturing folder with all gbr files is also provided.
Bill of Materials is also included.
 
Please reference Hackster.io page (https://www.hackster.io/ejm9/pocketbeagle-bop-it-project-39caf6)
    for more information on this project.
Link to Guthub
    - https://github.com/erickmor/ENGI301
